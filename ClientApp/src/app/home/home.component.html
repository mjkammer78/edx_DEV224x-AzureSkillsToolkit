Building Your Azure Skills Toolkit Demo Application

This app was built as part of a lab exercise from edX DEV224x course. It showcases the following elements:

  • an application with ASP.NET Core backend and Angular frontend
  • being deployed via CI to Azure as a Web App (PaaS)
  • integration with Azure Storage
  • secure design by default ;)
  • integration with Cognitive Services (Bing Search, Computer Vision)
  • integration with Azure Search
